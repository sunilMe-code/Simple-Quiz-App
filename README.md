🧠 Simple Quiz App

A responsive and interactive Quiz App built using HTML, CSS, and Vanilla JavaScript. This application allows users to answer multiple-choice questions.

The app dynamically updates questions and tracks user answers using JavaScript, making it a great project to understand DOM manipulation, event handling, and working with arrays/objects.

This project is ideal for beginners who want to build real-world frontend applications and improve their JavaScript logic skills.

✨ Features

❓ Multiple-choice questions

✅ Instant feedback on answers

🔄 Next question & restart option

📱 Fully responsive design

🛠️ Tech Stack

HTML5 – Structure

CSS3 – Styling
